### 构建镜像，在docker中运行

```bash
./run.sh
```
访问地址：http://localhost:8081/

### 本地运行

```
npm install
npm run start
```
访问地址：http://localhost:8080/

### Eslint

```
npm run eslint
```

### Build

```
npm run build
```


部署在 aws 服务上 访问： http://www.ilvwebsite.com.s3-website-us-east-1.amazonaws.com/

### Test

```
npm run test
```
